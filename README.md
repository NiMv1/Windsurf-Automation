# Windsurf Automation v4.0

🌊 **Самоулучшающаяся система** автоматизации Windsurf IDE с AI Brain.

## 🎯 Возможности

### 🤖 Automation
- **Поиск окон Windsurf** - Автоматически находит открытые окна
- **Новая сессия** - Открывает новое окно + Cascade + выбор модели
- **Отправка промптов** - Из файлов в папке промптов

### 🧠 AI Brain (Self-Improving)
- **Анализ кода** - Сканирует проект и находит улучшения
- **Автоматическое улучшение** - Исправляет код через AI (Groq/Ollama)
- **История изменений** - Отслеживает все улучшения
- **Тестирование** - Проверяет изменения перед применением

### 📊 Dashboard
- **System Monitor** - CPU, RAM, GPU, Disk в реальном времени
- **Project Stats** - Файлы, строки, функции, TODO
- **AI Stats** - Задачи, успешность, история
- **Logs** - Все события с цветовой подсветкой

## 🔄 Как работает самоулучшение

```
┌─────────────────────────────────────────────────────┐
│                   AI Brain Loop                      │
├─────────────────────────────────────────────────────┤
│  1. Scan → Анализ кода, поиск проблем               │
│  2. Plan → Создание задач улучшения                 │
│  3. Execute → AI генерирует исправления             │
│  4. Test → Проверка синтаксиса и тестов             │
│  5. Apply → Применение изменений                    │
│  6. Repeat → Переход к следующей задаче             │
└─────────────────────────────────────────────────────┘
```

## 📁 Структура

```
Windsurf-Automation/
├── main.py              # GUI с вкладками
├── config.json          # Настройки
├── requirements.txt     # Зависимости
├── run.bat             # Быстрый запуск
├── src/
│   ├── core.py          # Config, WindsurfController
│   ├── ai_brain.py      # 🧠 AI Brain, Tasks, Providers
│   └── system_monitor.py # 📊 CPU/RAM/GPU мониторинг
├── tasks/               # Задачи улучшения
├── prompts/             # Промпты
└── logs/                # Логи
```

## 🛠️ Установка

```bash
git clone https://github.com/NiMv1/Windsurf-Automation.git
cd Windsurf-Automation
pip install -r requirements.txt
```

## 🚀 Запуск

```bash
python main.py
```

## 🔌 AI Providers

| Provider | Тип | Модель | Бесплатно |
|----------|-----|--------|-----------|
| **Groq** | API | llama-3.1-70b | ✅ Да |
| **Ollama** | Local | llama3.1, mistral | ✅ Да |

### Получить Groq API Key:
1. Зайти на https://console.groq.com
2. Создать аккаунт
3. Скопировать API Key
